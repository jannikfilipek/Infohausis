

Kirchendach-Berechnungsprogramm
Das Kirchendach-Berechnungsprogramm ist ein Java-Programm, das die Oberfläche von Kirchendächern mit einer quadratischen Grundfläche und einer achteckigen Zwischenstufe berechnen kann.

Anleitung
Starten Sie das Programm, indem Sie die Test.java-Datei ausführen.

Geben Sie die geforderten Werte ein, wenn Sie dazu aufgefordert werden:

Länge der quadratischen Grundfläche
Länge einer Seite des Achtecks
Abstand des Achtecks zur Grundfläche
Gesamthöhe des Dachs
Überprüfen Sie die eingegebenen Werte und bestätigen Sie, ob sie korrekt sind.

Das Programm berechnet die Gesamtfläche des Kirchendachs ohne die Grundfläche und gibt das Ergebnis aus.

Das Programm endet.
